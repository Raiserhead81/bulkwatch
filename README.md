# BulkWatch — Bulk Carrier Intelligence

AI-powered database for bulk carrier ships worldwide with:
- Live AIS positions (planned)
- Real ship photos from Wikimedia Commons
- Specs (DWT, length, beam, draft, year built, builder, flag)
- AI price estimation based on age, type, tonnage
- Buy/Hold/Sell recommendation
- World map with all ships
- Comparison tool (2-5 ships side by side)
- Personal watchlist

## Tech Stack
- Next.js 16 (App Router)
- TypeScript 5
- Tailwind CSS 4 + shadcn/ui
- Leaflet (interactive world map)
- z-ai-web-dev-sdk (AI features)
